# 寺務管理システム (Temple Administration System)

寺院の寺務（お参り・檀家・過去帳・年忌・予定）を一元管理するスタッフ向け Web アプリのプロトタイプです。
claude.ai/design で作成したデザインをローカルで動作する形に実装したものです。

## 技術構成

ビルド不要の構成です。ブラウザ上で React 18 + Babel Standalone により JSX をその場でコンパイルして動作します。

- `index.html` — エントリポイント（React / ReactDOM / Babel を CDN から読み込み、各 `.jsx` / `.css` を順に読み込む）
- `tokens.css` — デザイントークン（色・タイポグラフィ・余白・影など）
- `styles.css` — 共通スタイル（ログイン画面・ダッシュボード・各画面の基本レイアウト）
- `notices.css` / `parish-map.css` / `settings.css` — 各画面固有のスタイル
- `sp.css` — スマートフォン（≤640px）向けレスポンシブ上書き

### 画面（コンポーネント）

| ファイル | 内容 |
| --- | --- |
| `app.jsx` | ルート（ログイン↔ダッシュボードの遷移、トースト、確認ダイアログ、Tweaks パネル） |
| `login.jsx` | ログイン画面（中央寄せ / 左右分割 / フルブリードの3バリアント） |
| `dashboard.jsx` | ログイン後のスタッフコンソール（サイドバー + 各画面の切替） |
| `visits.jsx` | お参り記録 |
| `parishioners.jsx` | 檀家管理（表 / カード / 境内マップの3バリアント） |
| `memorial.jsx` | 過去帳 |
| `schedule.jsx` | 予定管理（日 / 週 / 月カレンダー、ドラッグ&ドロップ移動） |
| `notices.jsx` | 年忌案内の自動集計・送付 |
| `parish-map.jsx` | 檀家地図 |
| `settings.jsx` | 設定（寺院情報・法要・予定種別・テンプレート・権限ほか） |
| `motifs.jsx` | 背景モチーフ（蓮 / 青海波 / 麻の葉）・ブランドアイコン |
| `tweaks-panel.jsx` | デザイン調整用の Tweaks パネル（フローティング） |

## 動かし方

Babel が `.jsx` を `fetch` で読み込むため、`file://` ではなくローカル HTTP サーバー経由で開く必要があります。

```bash
# どれか1つ
python3 -m http.server 8123
# または
npx serve -l 8123
```

ブラウザで <http://localhost:8123/> を開きます。

## メモ

- 本プロトタイプはデータをサーバーに保存しません。表示データはコンポーネント内のサンプルデータです。
- 画面右下の Tweaks パネル（デザインツール上で有効化されるもの）はレイアウトのバリアントやアクセントカラーを切り替えるための開発用 UI です。
